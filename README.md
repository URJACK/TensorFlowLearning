# TensorFlowLearning

## 相关命令

在nn、cnn、gan文件夹下都有完整、独立的demo运行示例，没有对其他文件的依赖。直接python命令运行指定的.py文件即可。

### 图的可视化

在graph文件夹下，使用下面的这段命令

```
tensorboard --logdir=.
```

即可启动tensorboard，在

```
localhost:6006
```

进行访问绘制好的图结构
